# dietscan
Quantification of dietary items from animal-derived metagenomes 

## Source databases

### BOLD

1. Login to BOLDSystems
2. Go to: https://bench.boldsystems.org/index.php/datapackages/Latest
3. Download data
```sh
curl -L -o BOLD_fasta.gz "https://bench.boldsystems.org/index.php/API_Datapackage/fasta?id=BOLD_Public.14-Mar-2025&uid=167dba260d5969"
```
